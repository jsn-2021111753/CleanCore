# Common

Shared utilities for the CleanCore artifact:

- `paths.py`: repository paths.
- `config.py`: YAML loading and merging.
- `seed.py`: random seed setup.
- `data.py`: NPZ data loading.
- `preprocessing.py`: train/test preprocessing.
- `model.py`: MLP classifier.
- `training.py`: training and prediction helpers.
- `metrics.py`: classification metrics.
- `timing.py`: progress and timing records.
- `results.py`: result writers.
- `interfaces.py`: method input/output dataclasses.
