# Lab2 Final Configs

This folder contains the selected final method configs for Lab2.

Datasets: wdbc, pendigits, magic, miniboone, skin, covertype.
Methods: cleancore, gradmatch, deem.

Each `dataset/method/method.yaml` file is the final config used by the Lab2 runner.

Corresponding result files are organized under `code/results/lab2` by dataset, noise type/rate, and method.
